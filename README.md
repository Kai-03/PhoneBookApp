# PhoneBook Web App

This is a web app which allows users to create, modify and/or delete contacts. Users are created in the registration page. Contacts are associated with different User Accounts and are retrieved using a unique ID from the User. This project demonstrates the CRUD operations in an MVC framework.

## Database

Two tables are created:

* Accounts which holds the User's login credentials
* Contacts which stores:
    * User contact info
    * User's contacts
